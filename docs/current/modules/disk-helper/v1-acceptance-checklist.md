# Disk Helper v1 验收清单

> 对照 [产品概要说明书_v1.md](./产品概要说明书_v1.md) AC-01～AC-13。  
> 手工验收：在 Tauri 桌面版（`VITE_USE_MOCK=false` 或 Tauri 运行时自动走真实 IPC）下逐项勾选。

**验收结论：v1 已通过（2026-06-23）**

## 功能验收

| 编号 | 场景 | 操作 | 预期 | 结果 |
|------|------|------|------|------|
| AC-01 | 磁盘总览 | 打开应用进入总览 | 展示 C 盘总/已用/可用；显示上次扫描时间 | ✅ |
| AC-02 | 空间浏览 | 点击「开始扫描」完成全量扫描 | 显示进度；完成后可浏览目录树与 Treemap | ✅ |
| AC-03 | 大文件 Top | 打开大文件 Top 榜单 | 按大小降序；可跳转目录 | ✅ |
| AC-04 | 风险分级 | 查看清理建议中某缓存目录 | 显示规则风险等级与说明 | ✅ |
| AC-05 | 软删除 | 选中安全项「移入隔离区」 | 文件进隔离区；日志有记录 | ✅ |
| AC-06 | 还原 | 隔离区选择项目「还原」 | 文件回到原路径；日志有记录 | ✅ |
| AC-07 | AI 问答 | 选中目录提问「能删吗」（本地 Ollama 或云端 Key） | 返回含风险与恢复建议的文本 | ✅ |
| AC-08 | 离线降级 | 断开网络后使用浏览/清理 | 本地功能正常；AI 提示不可用或失败可理解 | ✅ |
| AC-09 | 危险项保护 | 对危险级项尝试清理 | 被阻止或需二次确认且默认取消 | ✅ |
| AC-10 | AI 不越权 | 仅阅读 AI 回复 | 不会自动执行删除 | ✅ |

## 非功能验收

| 编号 | 类别 | 标准 | 结果 |
|------|------|------|------|
| AC-11 | 性能 | Top 100 大文件列表 ≤ 1s（完成扫描后） | ✅ |

## 发布前检查

| 项 | 命令/步骤 | 结果 |
|----|-----------|------|
| 全量扫描 | 本机 C 盘执行一次全量扫描 | ✅ |
| 双 AI | 本地 Ollama + 云端 DeepSeek 各问一句 | ✅ |
| 清理闭环 | 软删除 → 隔离区 → 还原 | ✅ |
| 打包 | `cd disk-helper && npm run tauri build` | ✅ |
| 安装运行 | 安装 `src-tauri/target/release/bundle/` 产物 | ✅ |

## 设置与 M6 专项

| 项 | 说明 | 结果 |
|----|------|------|
| 设置持久化 | 主题/隔离区/阈值/AI 模式保存后重启仍生效 | ✅ |
| 清空索引 | 设置 → 高级 → 输入「清空索引」确认；需重新扫描 | ✅ |
| 跨页上下文 | 浏览/清理带入 AI 后切换页面再回来，session 内上下文仍在 | ✅ |
| ErrorBoundary | 故意触发前端错误时显示友好页（可选） | ✅ |
