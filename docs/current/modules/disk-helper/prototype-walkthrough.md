# M1 UI 原型体验指南

> 运行：`cd disk-helper && npm run dev` → 打开 http://localhost:1420  
> 或在 Tauri 窗口：`npm run tauri dev`

## 建议体验路径

1. **总览** — 查看 C 盘容量告警、模拟扫描（开始/暂停/继续/取消）
2. **浏览** — 左树 + Treemap 联动；试大文件 Top、搜索「temp」
3. **清理** — 全选安全项 → 执行清理 → 跳转隔离区确认
4. **分析** — 从浏览页「问 AI」带入上下文，发送问题
5. **设置** — 切换 AI 模式（Ollama / DeepSeek）、深色主题
6. **操作日志** — 设置页链接进入，查看模拟审计记录

## 说明

- 全部为 **Mock 数据**，不扫描真实 C 盘
- 清理/隔离/AI 回复均为前端模拟
- 确认 UI 满意后再进入 M2 对接 Rust 后端
